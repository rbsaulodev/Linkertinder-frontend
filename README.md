# Linketinder — Frontend MVP

Interface frontend do **Linketinder**, um sistema de conexão entre candidatos e empresas, inspirado na lógica de match do Tinder.

---

## Tecnologias utilizadas

### Frontend
- **HTML5** — estrutura semântica das páginas
- **CSS3** — design system com variáveis CSS, animações, layout responsivo
- **TypeScript** (via ES Modules inline) — lógica de classes, decorators, tipagem
- **Chart.js** (CDN) — gráfico de barras de competências por empresa
- **Google Fonts** — tipografia: Syne + DM Mono
- **LocalStorage** — persistência de dados no navegador

## Como executar o Frontend

Siga os passos abaixo para configurar e rodar o ambiente de desenvolvimento:

1.  **Clonar o repositório:**
    ```bash
    git clone https://github.com/rbsaulodev/Linkertinder-frontend.git
    ```
2.  **Acessar o diretório:**
    ```bash
    cd Linkertinder-frontend
    ```
3.  **Instalar as dependências:**
    > Este passo é essencial para baixar as bibliotecas necessárias (como o Vite ou React).
    ```bash
    npm install
    ```
4.  **Iniciar o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```
5.  **Acessar a aplicação:**
    Após rodar o comando, o terminal exibirá um link (geralmente `http://localhost:5173/`). Copie e cole no seu navegador.
